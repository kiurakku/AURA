# 🎰 AURA Casino - Telegram Mini App

Повнофункціональне казино для Telegram з підтримкою криптовалютних платежів, ігор та реферальної системи.

## 🌟 Особливості

- ✅ **Мультимовність** - 6 мов (Українська, Російська, Англійська, Китайська, Німецька, Іспанська)
- ✅ **Онлайн ігри** - змагайся з іншими гравцями в реальному часі
- ✅ **Прив'язка гаманців** - до 3 гаманців одночасно
- ✅ **Налаштування конфіденційності** - повний контроль над даними
- ✅ **Правові документи** - політика конфіденційності, правила платформи
- ✅ **Digital Luxury UI** - преміальний дизайн з неоновими ефектами
- ✅ **Provably Fair** - чесні ігри з можливістю перевірки

## 🚀 Швидкий старт

### Встановлення

```bash
npm run install:all
```

### Запуск

**Backend:**
```bash
cd backend
npm run dev
```

**Frontend:**
```bash
cd frontend
npm run dev
```
